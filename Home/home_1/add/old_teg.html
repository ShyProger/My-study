Тег <tt> отображает текст моноширинным текстом

Тег <code> предназначен для отображения одной или нескольких строк текста, который представляет собой программный код.
print("hello world")

Тег <kbd> используется для обозначения текста, который набирается на клавиатуре или для названия клавиш: Для быстрого закрытия программы используйте комбинацию клавиш: <Alt>+<F4>

Тег <var> используется для выделения переменных компьютерных программ.
Переменная text возвращает фразу "Hello world"

Тег <sub> отображает шрифт в виде нижнего индекса: H2O
Тег <sup> отображает шрифт в виде верхнего индекса: E=mc2

Тег <blockquote> предназначен для выделения длинных цитат внутри документа.

Lorem ipsum dolor sit amet, consectetuer adipiscing elit, sed diem nonummy nibh euismod tincidunt ut lacreet dolore magna aliguam erat volutpat. Ut wisis enim ad minim veniam, quis nostrud exerci tution ullamcorper suscipit lobortis nisl ut aliquip ex ea commodo consequat.

Тег <q> используется для выделения в тексте цитат.Содержимое контейнера автоматически отображается в браузере в кавычках.
Мальчик сказал: И так сойдёт.

Тег <ins> предназначен для выделения текста, который был добавлен в новую версию документа.
Тег <del> используется для выделения текста, который был удален в новой версии документа.

Тег <address> предназначен для хранения информации об авторе и может включать в себя любые элементы HTML вроде ссылок, текста, выделений и т.д.
