Форматы файлов

.doc или .DOC (от англ. Document) — расширение имени файла, используемое для файлов, представляющих текст, с разметкой или без. Расширение «.doc» часто использовалось для обозначения текстовых файлов, содержащих текст без форматирования, позже стало использоваться для двоичных форматов с разметкой. Документ MS Word

Portable Document Format (PDF) — межплатформенный открытый формат электронных документов, изначально разработанный фирмой Adobe Systems с использованием ряда возможностей языка PostScript. Документ PDF

ZIP - популярный формат сжатия данных и архивации файловю Фйал в этом формате обычно имеет расширение .zip и хранит в сжатом или несжатом виде один или несколько файловБ которые можно из него извлечь путём распаковки с помощью специальной программы. Архив
